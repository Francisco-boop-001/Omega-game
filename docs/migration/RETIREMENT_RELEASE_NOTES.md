# Legacy Runtime Retirement Notes

Updated: 2026-02-06

## Policy Statement

- Rust runtime path is the default release path.
- Legacy C runtime release grace path is retired from workflow policy.
- Legacy C runtime is no longer part of release workflow controls.

## Support Policy

- Active support applies to Rust runtime paths only.
- Legacy runtime receives no new feature or compatibility guarantees after retirement date.
